referencia/ — arquivo histórico (não deployado)
==============================================

Esta pasta guarda materiais de referência offline que **não** fazem parte do
app Pontuação Conclave nem são publicados no GitHub Pages.

  planilhas/   — planilhas Excel legadas de pontuação (ver planilhas/README.txt)
  documentos/  — ordens de culto, regulamentos em .docx, etc.
  scripts/     — utilitários legados (ex.: gerar_ordem_culto.py)

O deploy em `.github/workflows/pages.yml` exclui `referencia/` do site publicado.
